0x03 Installation

Clone the repository

$ git clone git@github.com:ZealDev/T3hb0t.git

Files & Usage

1x01 commands_list.py

This is the file where you put all your commands. All public methods (without a _ infront) will be available through the bot by doing: $methodName <arg1> <arg2> . All private methods (with _ infront of method name) will not be accessible through the bot, but will ofcourse be accessible in the class.

The commands usage is documented in the docstring of the method. This is then used when you do $help to get a list of commands and how to use them.

Standard commands are:

$help

Outputs the available commands and their usage.

$note : Usage: note <to nickname> <note>

If <to nickname> <note> is supplied, it saves a note to the given nickname. If <to nickname> <note> is omitted, it lists out all notes that are not set to 'read'.

$notelist : Usage: notelist

Outputs a list of all notes with their corresponding id's.

$noteread : Usage: noteread <id>

Gives a note the status of 'read', so it doesn't show in $note.

$backin : Usage: backin <time>

Saves a your expected time of return. Used like this:

$backin 2.5h (for 2 and a half hours)

$backin 30m (for 30 minutes)

$back : Usage: back <nickname>

Outputs when the given nickname (person) will be back (if the person saved it with $backin).
